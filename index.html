MAPA INTERACTIU

Per què s'ha creat aquest mapa interactiu?
El mapa interactiu s'ha creat amb l'objectiu de presentar els resultats de l'estudi d'una manera més visual i intuïtiva. Després de recollir i analitzar totes les dades sobre els nivells de soroll a Taradell, es va considerar necessari trobar una forma de representar aquesta informació que fos més atractiva i fàcil de comprensió per a tothom.

D'aquesta manera

Com s'ha creat el mapa interactiu?
Per a la implementació del mapa interactiu s'ha utilitzat la llibreria JavaScript de codi obert Leaflet. Després d'avaluar diferents opcions, es va optar per Leaflet per la seva gran personalització que ofereix i la facilitat d'ús.

Mitjançant les funcions de Leaflet, s'ha creat un mapa bàsic de Taradell i s'han afegit elements com marcadors, i línies per representar els diferents punts de mesura i les propostes de millora. També s'ha afegit interactivitat, de manera que en fer clic sobre cada marcador apareix informació com la mitjana de soroll del punt.

Per compartir el mapa interactiu s'ha publicat en una pàgina web utilitzant la plataforma GitHub, que permet allotjar el codi JavaScript i visualitzar el mapa de forma senzilla en qualsevol navegador web estàndard.
